Control Flow

**if else**

IF - this block will Executes when its true

Else - IF block fails means this block will execute.

**Multiple if else:**

Contains of multiple IF Else blocks.

Used when have Multiple conditions.

**Switch Case:**

Used when we have many conditions.

Every Switch will have Case and Break and  every switch  will have a default case.

Syntax :

switch(condition)

{

case 1: 

//Print;

break;

case 2: 

//Print;

break;

default: 

//break;

}
